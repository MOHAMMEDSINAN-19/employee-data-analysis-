# Employee Data Analysis using Python

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the **ABC Company Employee Dataset** using Python. The objective is to clean the dataset, analyze employee information, create meaningful visualizations, and extract valuable business insights.

The project demonstrates the use of **Pandas**, **NumPy**, and **Matplotlib** for data preprocessing, analysis, and visualization.

---

## 📂 Dataset

- **Dataset:** ABC Company Employee Dataset
- **Rows:** 458
- **Columns:** 9

---

## 🎯 Objectives

- Preprocess the employee dataset.
- Replace the **Height** column with random values between **150 and 180**.
- Analyze employee distribution across teams.
- Analyze employee positions.
- Identify the predominant age group.
- Find the team and position with the highest salary expenditure.
- Analyze the relationship between age and salary.
- Present results using charts and graphs.

---

## 🛠 Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib

---

## 📋 Project Tasks

### 1. Data Preprocessing
- Loaded the dataset.
- Checked for missing values.
- Replaced the **Height** column with random values ranging from **150–180**.
- Verified data consistency.

### 2. Employee Distribution by Team
- Calculated the number of employees in each team.
- Calculated percentage distribution.
- Created a bar chart.

### 3. Employee Distribution by Position
- Grouped employees by their positions.
- Displayed the count for each position.
- Created a bar chart.

### 4. Age Group Analysis
- Categorized employees into age groups.
- Identified the predominant age group.
- Created a visualization.

### 5. Salary Expenditure Analysis
- Calculated total salary expenditure by team.
- Calculated total salary expenditure by position.
- Identified the highest salary spending team and position.
- Visualized the results.

### 6. Correlation Analysis
- Calculated the correlation between Age and Salary.
- Created a scatter plot to visualize the relationship.

---

## 📊 Visualizations

The project includes:

- Team-wise Employee Distribution
- Team Percentage Distribution
- Position-wise Employee Distribution
- Age Group Distribution
- Salary Expenditure by Team
- Salary Expenditure by Position
- Age vs Salary Scatter Plot

---

## 📈 Key Insights

- Employee distribution varies across different teams.
- Some positions have significantly more employees than others.
- A specific age group represents the majority of employees.
- Salary expenditure is concentrated in a few teams and positions.
- The Age vs Salary analysis helps understand whether employee age influences salary.
